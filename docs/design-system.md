# Design System — Distribuidora Shon C.A.

## Color Palette

| Role                   | Value                  | Usage                                     |
|------------------------|------------------------|-------------------------------------------|
| Background/Base        | `#FFFFFF` or `#F9F9F9` | Main background, clean look               |
| Accent/Buttons         | `#E91E63` or `#FF4081` | CTAs, highlights, "cute" details          |
| Primary Text           | `#212121`              | Body text, titles                         |
| Secondary Text/Borders | `#757575`              | Subtitles, borders, placeholders          |
| Success (Offers)       | `#4CAF50`              | Discounts/offers badges (use sparingly)   |

## Typography

| Role           | Font                  | Weight       |
|----------------|-----------------------|--------------|
| Headers/Titles | Montserrat or Poppins | Bold (700)   |
| Body Text      | Inter or Poppins      | Regular (400)|
| Labels/Prices  | Quicksand             | Medium (500) — artisanal/handmade feel |

> Import from Google Fonts: Montserrat, Poppins, Inter, Quicksand

## UI Style — Soft UI / Modern Minimalism

- **Border radius:** `12px` – `16px` on cards and buttons (no sharp edges)
- **Shadows:** `box-shadow: 0 4px 20px rgba(0,0,0,0.05);` — subtle depth
- **Animations:** Slide-in from right when cart opens
- **Primary CTA button:** Large, `#E91E63` background, rounded corners, hover darkens slightly

## Cart Layout Requirements

Each product card must show:
1. **Thumbnail** — small clear product image
2. **Personalization details** — secondary text below title (e.g. "Mickey & Minnie", uploaded photo name)
3. **Quantity selector** — clean `+` / `-` buttons
4. **Shipping summary** — simple shipping calculator or "Free shipping over S/..." message (Peruvian market: Arequipa / Lima)

## Brand Notes

- Shon sells personalized embroidered products (photos, names on garments)
- Neutral base so product colors stand out
- Curves and softness over sharp edges — reflects the brand's warmth
- Pink accent evokes love, couples, and "cute" details
